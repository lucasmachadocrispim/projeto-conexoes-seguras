# Projeto Conexões Seguras

Projeto semestral desenvolvido para a disciplina de Segurança em Redes.

O objetivo deste projeto é analisar, demonstrar e mitigar vulnerabilidades críticas na camada de aplicação e no tráfego de dados em redes corporativas, cobrindo desde a interceptação de pacotes até a proteção de sessões de usuário.

--------------------------------------------------------------------------------------

## 🛠️ Estrutura do Repositório

O projeto está dividido em módulos práticos e teóricos, organizados da seguinte forma:

`modulo1-sniffing`: Estudo e demonstração de interceptação de tráfego de dados em texto plano utilizando ferramentas de análise de pacotes (como Wireshark).
`modulo2-tls`: Implementação e análise do impacto dos protocolos TLS/SSL na cifragem do tráfego, garantindo confidencialidade e integridade através do HTTPS.
`modulo3-sessao`: Prova de Conceito (PoC) focada em segurança de aplicação, demonstrando ataques de injeção de script (XSS), roubo de `SessionID` e a mitigação através da flag `HttpOnly`.
`evidencias`: Pasta centralizada contendo os prints de telas, logs e relatórios que comprovam a execução bem-sucedida de todos os testes laboratoriais.
