Projeto Conexões Seguras

Sobre o Projeto

Este projeto semestral foi desenvolvido para analisar o tráfego de dados e aplicar conceitos práticos de segurança na camada de aplicação. Investigamos desde a interceptação de pacotes em texto claro até técnicas avançadas de proteção de sessões web para mitigar ataques cibernéticos comuns em redes corporativas.

Módulos do Projeto

Navegue pelas implementações e laboratórios práticos executados pelo grupo:

Módulo 1: Sniffing de Rede

Análise e interceptação de tráfego de dados em texto plano utilizando ferramentas de monitoramento de pacotes (Wireshark).

Ver Relatório no GitHub

Módulo 2: Cifragem com TLS

Implementação e validação de segurança através do protocolo HTTPS/TLS para garantir confidencialidade do tráfego.

Ver Relatório no GitHub

Módulo 3: Segurança de Sessão (PoC)

Prova de Conceito interativa demonstrando o roubo de sessão via script (XSS) e a mitigação com a flag HttpOnly.

Executar PoC do Módulo 3
