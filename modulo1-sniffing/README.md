# Módulo 1 - Sniffing e Interceptação (Camadas 2 e 3)

**Ferramenta usada:** Wireshark

# Passo a Passo Resumido
1. Configuramos a interface de rede para o modo promíscuo.
2. Iniciamos a captura de tráfego usando o Wireshark.
3. Filtramos e analisamos os pacotes de interesse (Camadas 2 e 3).
4. Paramos a captura e salvamos o arquivo `.pcapng`.

# Como abrir o arquivo do Wireshark
Esse repositório contém a Prova de Conceito (PoC) prática deste módulo. Para visualizar:
1. Certifique-se de ter o Wireshark (https://www.wireshark.org/) instalado na sua máquina.
2. Baixe o arquivo `captura.pcapng` que está nesta mesma pasta.
3. Dê um duplo clique no arquivo baixado, e ele abrirá automaticamente no Wireshark com todos os pacotes capturados.

#  Resultado Obtido
A captura foi realizada com sucesso. As evidências visuais (prints da captura, dos pacotes e da interface em modo promíscuo) estão salvas na pasta raiz em: `evidencias/modulo1/`.
