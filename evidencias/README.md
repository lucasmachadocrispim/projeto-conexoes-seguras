# Evidências do Projeto

Esta pasta centraliza todas as comprovações práticas, relatórios, logs e capturas de tela (prints) gerados durante a execução dos laboratórios de cada módulo.

## 📁 Organização Interna

A pasta está dividida de acordo com os módulos do projeto semestral:

* **`/modulo1`**: Prints da interceptação de pacotes em texto plano (Wireshark).
* **`/modulo2`**: Comprovações da cifragem de dados e validação do protocolo TLS/HTTPS.
* **`/modulo3`**: Capturas de tela da Prova de Conceito (PoC) simulando o ataque XSS e a proteção do HttpOnly.
